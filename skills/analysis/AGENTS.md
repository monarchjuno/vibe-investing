# Analysis Category Guide

## Scope

- Apply these rules to `skills/analysis/` and its descendants unless a deeper file overrides them.

## Purpose

- Store cross-asset analytical skills that can be adapted to equities, crypto, credit, commodities, or other investable assets.

## Include Here

- Fundamental analysis
- Valuation analysis
- Risk analysis
- Market structure analysis

## Rules

- Prefer one reusable analytical skill with clear asset-class branches over many near-duplicate asset-specific skills.
- Put asset-specific heuristics in `references/` when the core workflow is shared.
- Make the target asset class explicit at the start of the workflow whenever interpretation rules differ.
